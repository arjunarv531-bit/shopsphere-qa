
# BUG-001 Login Button Becomes Unresponsive

## Bug Summary

Login button becomes unresponsive after three invalid login attempts.

## Environment

* Browser: Chrome
* Version: Latest
* OS: Windows 11

## Steps to Reproduce

1. Open Login Page
2. Enter valid email
3. Enter wrong password
4. Click Login
5. Repeat 3 times

## Expected Result

System should display an error message and allow the user to retry.

## Actual Result

Login button becomes unresponsive and user cannot attempt login again.

## Severity

High

## Priority

Medium

## Status

Open
