# ShopSphere QA Suite

## About Project

ShopSphere QA Suite is my QA Automation portfolio project.

The goal of this project is to demonstrate:

* Manual Testing
* Test Case Design
* Bug Reporting
* API Testing using Postman
* SQL Validation
* Jira Issue Tracking
* Playwright Automation Testing
